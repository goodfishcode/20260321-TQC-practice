# TQC+ 程式語言 Python 第8類：字串(String)的運作（801–810）

技能內容：字串的建立、字串的內建函式、字串的應用

---

# 801 字串索引

## 題目說明

請撰寫程式，輸入一個字串並輸出指定索引位置的字元。

## 輸入說明

輸入一個字串。

## 輸出說明

輸出指定索引位置的字元。

## 範例輸入

```
Hello
```

## 範例輸出

```
H
```

---

# 802 字元對應

## 輸入說明

輸入一個字串。

## 輸出說明

輸出字元與其索引。

## 範例輸入

```
Python
```

## 範例輸出

```
0 P
1 y
2 t
3 h
4 o
5 n
```

# 809 密碼規則

## 題目說明

請撰寫程式，檢查密碼是否符合規則：

- 至少8個字元
- 包含大寫字母
- 包含小寫字母
- 包含數字

## 輸入說明

輸入一個密碼字串。

## 輸出說明

符合規則輸出 True，否則 False。

## 範例輸入

```
Abc12345
```

## 範例輸出

```
True
```
