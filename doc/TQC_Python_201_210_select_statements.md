# TQC+ 程式語言Python 第2類：選擇敘述（201–210 題目整理）

## 201 偶數判斷
題目說明：請開啟 PYD201.py 檔案，依題意進行作答，判斷輸入值是否為偶數；作答完成請另存新檔為 PYA201.py 再進行評分。  
設計說明：輸入一個正整數，判斷是否為偶數（even）。  
輸入說明：一個正整數  
輸出說明：判斷是否為偶數  

範例輸入1：
```
56
```
範例輸出1：
```
56 is an even number.
```

範例輸入2：
```
21
```
範例輸出2：
```
21 is not an even number.
```

## 207 折扣方案
題目說明：請開啟 PYD207.py 檔案，依題意進行作答，判斷輸入值之折扣並計算實付金額；作答完成請另存新檔為 PYA207.py 再進行評分。  
設計說明：輸入購物金額（需 >= 8000），依折扣方案計算實付金額：  
- 8000（含）以上：9.5折  
- 18000（含）以上：9折  
- 28000（含）以上：8折  
- 38000（含）以上：7折  
輸入說明：一個數值，需大於 8000（含）以上  
輸出說明：折扣優惠後的實付金額（輸出不需指定小數點位數）  

範例輸入：
```
12000
```
範例輸出：
```
11400.0
```

## 210 三角形判斷
題目說明：請開啟 PYD210.py 檔案，依題意進行作答，檢查輸入值是否可組成三角形；作答完成請另存新檔為 PYA210.py 再進行評分。  
設計說明：輸入三個邊長，檢查是否可組成三角形；可組成則輸出周長，否則輸出 `Invalid`。檢查方法：任意兩邊長之和大於第三邊長。  
輸入說明：三個正整數  
輸出說明：可以組成三角形則輸出周長；否則輸出 Invalid  

範例輸入1：
```
5
6
13
```
範例輸出1：
```
Invalid
```

範例輸入2：
```
1
1
1
```
範例輸出2：
```
3
```
