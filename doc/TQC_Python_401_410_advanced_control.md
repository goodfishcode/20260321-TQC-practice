# TQC+ 程式語言Python 第4類：進階控制流程（401–410 題目整理）

## 401 最小值
題目說明：輸入十個數值，找出最小值並輸出。  
設計說明：由使用者輸入十個數字，找出最小值。  
輸入說明：十個數值  
輸出說明：十個數值中的最小值  
範例輸入：
23 57 48 2 99 70 9 65 35 88  
範例輸出：
2

## 402 不定數迴圈-最小值
題目說明：輸入不定數量的數值，找出最小值並輸出，直到輸入 9999 結束。  
設計說明：使用 while 迴圈持續輸入數值，遇到 9999 停止並輸出最小值。  
輸入說明：不定數量的整數（以 9999 結束）  
輸出說明：最小值  
範例輸入：
12 74 2 67 9999  
範例輸出：
2

## 403 倍數總和計算
題目說明：輸入一個正整數 n，計算從 1 到 n 之間所有 3 的倍數之總和。  
設計說明：使用迴圈累加倍數。  
輸入說明：一個正整數 n  
輸出說明：總和  
範例輸入：
15  
範例輸出：
45

## 404 數字反轉判斷
題目說明：輸入一個數字，將其反轉後輸出。  
設計說明：輸入正整數，利用迴圈反轉。  
輸入說明：一個正整數  
輸出說明：反轉數字  
範例輸入：
1234567  
範例輸出：
7654321

## 405 不定數迴圈-分數等級
題目說明：輸入不定數量的分數，依等級規則輸出 A/B/C/D/E，直到輸入 -1 結束。  
設計說明：使用 while 迴圈讀取，按分數區間判斷等級。  
輸入說明：不定數量的分數（以 -1 結束）  
輸出說明：每筆分數對應的等級  
範例輸入：
80 59 90 -1  
範例輸出：
A E A

## 406 不定數迴圈-BMI計算
題目說明：輸入不定數量的身高/體重，計算 BMI 並輸出判斷結果，直到輸入 0 結束。  
設計說明：使用 while 迴圈；BMI = 體重/身高^2（身高公尺）；依範圍輸出體重判定。  
輸入說明：不定數量的兩個浮點數（身高、體重），以 0 結束  
輸出說明：BMI 及判定結果  
範例輸入：
1.73 50 0  
範例輸出：
(略)

## 407 不定數迴圈-閏年判斷
題目說明：輸入不定數量年份，判斷是否閏年；直到輸入 0 結束。  
設計說明：依閏年規則判斷；輸出 “YYYY is a leap year.” 或 “YYYY is not a leap year.”  
輸入說明：不定數量年份，以 0 結束  
輸出說明：判斷結果  
範例輸入：
2012 2009 0  
範例輸出：
2012 is a leap year.  
2009 is not a leap year.

## 408 奇偶數個數計算
題目說明：輸入不定數量整數，統計奇數與偶數個數，直到輸入 0 結束。  
設計說明：使用 while 迴圈累計奇偶數個數，最後輸出奇數與偶數數量。  
輸入說明：不定數量整數，以 0 結束  
輸出說明：奇數與偶數個數  
範例輸入：
3 4 9 0  
範例輸出：
odd: 2  
even: 1

## 409 得票數計算
題目說明：輸入不定數量整數代表投票（1、2、3、4），直到輸入 0 結束；計算並輸出各候選人得票數。  
設計說明：使用 while 迴圈統計各票數。  
輸入說明：不定數量票數，以 0 結束  
輸出說明：各候選人得票數  
範例輸入：
2 3 4 1 0  
範例輸出：
A: 1 B: 1 C: 1 D: 1

## 410 繪製等腰三角形
題目說明：輸入高度 h，輸出由星號 (*) 組成的等腰三角形。  
設計說明：輸入正整數 h；每列先輸出空白對齊，再輸出 2*i-1 個星號（或題意要求的符號排列）。  
輸入說明：一個整數 h  
輸出說明：等腰三角形  
範例輸入：
3  
範例輸出：
  *  
 ***  
*****
