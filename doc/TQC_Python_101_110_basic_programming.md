## 109 正五邊形面積計算

題目說明：請開啟 PYD109.py 檔案，依題意進行作答，計算正五邊形面積；作答完成請另存新檔為 PYA109.py 再進行評分。  
設計說明：輸入正數 `s`（邊長），計算並輸出正五邊形面積 Area。提示1：建議使用 `import math` 的 `math.pow` 及 `math.tan`；提示2：正五邊形面積公式：`Area = (5 * s^2) / (4 * tan(pi/5))`；提示3：輸出到小數點後第 4 位。  
輸入說明：正數 s（邊長）  
輸出說明：面積（Area）  
範例輸入：
5  
範例輸出：
Area = 43.0119

## 110 正n邊形面積計算

題目說明：請開啟 PYD110.py 檔案，依題意進行作答，計算正 n 邊形面積；作答完成請另存新檔為 PYA110.py 再進行評分。  
設計說明：輸入兩個正數 `n`、`s`，其中 `n` 代表邊數、`s` 代表邊長；計算並輸出正 n 邊形面積 Area。提示1：建議使用 `import math` 的 `math.pow` 及 `math.tan`；提示2：正 n 邊形面積公式：`Area = (n * s^2) / (4 * tan(pi/n))`；提示3：輸出到小數點後第 4 位。  
輸入說明：正數 n、s  
輸出說明：正 n 邊形面積  
範例輸入：
8  
6  
範例輸出：
Area = 173.8234
