# TQC+ 程式語言 Python 第7類：數組（Tuple）、集合（Set）、詞典（Dictionary）

技能內容：數組、集合、詞典的建立、運作及應用

---

# 701 串列數組轉換

## 題目說明

請撰寫程式，將輸入的串列資料轉換為數組（Tuple）並輸出。

## 輸入說明

輸入一列整數（以空白分隔）。

## 輸出說明

將串列轉換為數組並輸出。

## 範例輸入

```
1 2 3 4 5
```

## 範例輸出

```
(1, 2, 3, 4, 5)
```

---

# 702 數組合併排序

## 題目說明

請撰寫程式，將兩個數組合併後進行排序並輸出。

## 輸入說明

輸入兩列整數。

## 輸出說明

將兩個數組合併後排序並輸出。

## 範例輸入

```
1 5 3
2 4 6
```

## 範例輸出

```
1 2 3 4 5 6
```

# 708 詞典合併

## 題目說明

請撰寫程式將兩個詞典合併並輸出。

## 輸入說明

輸入兩列 key:value 資料。

## 輸出說明

輸出合併後的詞典。

## 範例輸入

```
a:1 b:2
c:3 d:4
```

## 範例輸出

```
{'a': 1, 'b': 2, 'c': 3, 'd': 4}
```

---

# 709 詞典排序

## 題目說明

請撰寫程式將詞典依照 key 進行排序並輸出。

## 輸入說明

輸入多組 key:value。

## 輸出說明

依 key 排序後輸出詞典。

## 範例輸入

```
b:2 a:1 c:3
```

## 範例輸出

```
a:1
b:2
c:3
```

---

# 710 詞典搜尋

## 題目說明

請撰寫程式依照 key 搜尋詞典中的值。

## 輸入說明

輸入詞典資料與查詢 key。

## 輸出說明

輸出對應的 value。

## 範例輸入

```
a:1 b:2 c:3
b
```

## 範例輸出

```
2
```
