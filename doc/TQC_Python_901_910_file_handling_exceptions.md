# TQC+ 程式語言 Python 第9類：檔案與異常處理（901–910）

技能內容：文字 I/O、檔案建立、資料讀寫、編碼（Encoding）、異常處理

---

# 901 成績資料

## 題目說明

請撰寫程式，將輸入的學生姓名與成績寫入檔案中，並輸出檔案內容。

## 輸入說明

輸入學生姓名與成績。

## 輸出說明

將資料寫入檔案並顯示檔案內容。

## 範例輸入

```
John 80
Mary 90
```

## 範例輸出

```
John 80
Mary 90
```

# 902 資料加總

## 題目說明

請撰寫程式，讀取檔案中的數值並計算總和。

## 輸入說明

輸入數值資料並寫入檔案。

## 輸出說明

輸出所有數值的總和。

## 範例輸入

```
10
20
30
```

## 範例輸出

```
60
```
